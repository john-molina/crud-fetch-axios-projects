# CRUD con Fetch API

Este proyecto fue realizado como práctica de aprendizaje siguiendo un curso de YouTube enfocado en desarrollo frontend con JavaScript y consumo de datos mediante Fetch API.

## Descripción

El proyecto implementa operaciones CRUD (Create, Read, Update, Delete) utilizando Fetch API para la gestión dinámica de datos. Durante el desarrollo se trabajó con manipulación del DOM, peticiones asíncronas y actualización de la interfaz sin necesidad de recargar la página.

Este trabajo fue replicado con fines educativos para reforzar conocimientos en JavaScript moderno y buenas prácticas de desarrollo frontend.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- JSON

## Funcionalidades

- Obtener datos desde un archivo JSON o API
- Mostrar información dinámicamente en pantalla
- Crear nuevos registros
- Editar registros existentes
- Eliminar registros
- Actualización automática de la interfaz
- Manejo de peticiones asíncronas

## Objetivo del proyecto

Fortalecer conocimientos en desarrollo frontend, consumo de APIs, asincronía en JavaScript y manipulación dinámica del DOM mediante la práctica guiada de un sistema CRUD funcional.

## Nota

Este proyecto fue desarrollado como parte de un proceso de aprendizaje práctico siguiendo contenido educativo de YouTube y posteriormente adaptado para reforzar habilidades técnicas personales.
